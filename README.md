## Slash-Worker

Serverless Discord slash commands bot for Cloudflare's Workers.

Note:
<br>
You need to define enviorment variables:
```
export NOMICS_API_TOKEN=
export PUBLIC_KEY=
```

Don't hesitate to create pull requests for any improvements.
